<p><b> Knowing this first </b> (<span class="_800000"><span lang="el-GR">τουτο πρωτον γινωσκοντες</span></span>). Present active participle of <span class="_800000"><span lang="el-GR">γινωσκω</span></span>. See <span class="scriptRef" com="rwp" lang="eng" ref="2pe+1:20">2 Peter 1:20</span> for this identical phrase. Nominative absolute here where accusative <span class="_800000"><span lang="el-GR">γινωσκοντας</span></span> would be regular. Peter now takes up the <span class="_800000"><span lang="el-GR">παρουσια</span></span> (<span class="scriptRef" com="rwp" lang="eng" ref="2pe+1:16">2 Peter 1:16</span>) after having discussed the <span class="_800000"><span lang="el-GR">δυναμις</span></span> of Christ.</p>
<p><b>In the last days </b> (<span class="_800000"><span lang="el-GR">επ' εσχατων των ημερων</span></span>). "Upon the last of the days." <span class="scriptRef" com="rwp" lang="eng" ref="jude+1:18">Jude 1:18</span> has it <span class="_800000"><span lang="el-GR">επ' εσχατου χρονου</span></span> (upon the last time). In <span class="scriptRef" com="rwp" lang="eng" ref="1pe+1:5">1 Peter 1:5</span> it is <span class="_800000"><span lang="el-GR">εν καιρω εσχατω</span></span> (in the last time), while <span class="scriptRef" com="rwp" lang="eng" ref="1pe+1:20">1 Peter 1:20</span> has <span class="_800000"><span lang="el-GR">επ' εσχατου των χρονων</span></span> (upon the last of the times). John has usually <span class="_800000"><span lang="el-GR">τη εσχατη ημερα</span></span> (on the last day, <span class="scriptRef" com="rwp" lang="eng" ref="2pe+6:39">2 Peter 6:39</span>). Here <span class="_800000"><span lang="el-GR">εσχατων</span></span> is a predicate adjective like <span class="_800000"><span lang="el-GR">συμμυς μονς</span></span> (the top of the mountain).</p>
<p><b>Mockers with mockery </b> (<span class="_800000"><span lang="el-GR">εμπαιγμονη εμπαικτα</span></span>). Note Peter's play on words again, both from <span class="_800000"><span lang="el-GR">εμπαιζω</span></span> (<span class="scriptRef" com="rwp" lang="eng" ref="mt+2:16">Matthew 2:16</span>), to trifle with, and neither found elsewhere save <span class="_800000"><span lang="el-GR">εμπαικτης</span></span> in <span class="scriptRef" com="rwp" lang="eng" ref="jdg+1:18">Judges 1:18</span>; <span class="scriptRef" com="rwp" lang="eng" ref="isa+3:4">Isaiah 3:4</span> (playing like children). </p>
