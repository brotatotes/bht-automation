<p>The heavens - which now are [<span class="_800000"><span lang="el-GR">οι νυν ουρανοι] </span></span>. A construction similar to the then world (ver. 6). The now heavens, or the present heavens. Kept in store [<span class="_800000"><span lang="el-GR">τεθησαυρισμενοι] </span></span>. Rev., stored up. Lit., treasured up. The same word which is used in <span class="scriptRef" com="vnt" lang="eng" ref="lu+12:21">Luke 12:21</span>, layeth up treasure. Sometimes with the kindred noun qhsaurouv, treasures, as <span class="scriptRef" com="vnt" lang="eng" ref="mt+6:19">Matthew 6:19</span>; lit., treasure treasures. </p>
<p>Unto fire. Some construe this with treasured up; as Rev., stored up for fire; others with reserved, as A. V.; others again give the sense stored with fire, indicating that the agent for the final destruction is already prepared.</p>
